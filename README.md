README: Программа для создания датированной папки и уведомления (C#-версия)

Назначение
Программа создаёт в каталоге своего расположения папку с именем в формате ДД.ММ.ГГГГ (текущая дата). 
После выполнения действия выводит уведомление.

Функциональные возможности
*	Автоматическое формирование имени папки на основе текущей даты (например, 16.01.2026).
*	Проверка существования папки:
        	если папка отсутствует — создаётся новая;
         	если папка уже есть — выводится предупреждение.
*	Визуальное уведомление через всплывающее окно с анимацией появления текста.

Принцип работы
1.	Определяется каталог, где расположен исполняемый файл.
2.	Формируется имя папки по шаблону ДД.ММ.ГГГГ.
3.	Проверяется наличие папки с таким именем в текущем каталоге.
4.	В зависимости от результата:
        	если папки нет — она создаётся, и показывается окно с сообщением «Папка создана: [путь]»;
        	если папка есть — показывается окно с предупреждением «Папка [имя] уже существует!».
